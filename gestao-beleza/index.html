Backup e Segurança dos Dados

ATENÇÃO: Seus dados são locais!

📱 Os dados ficam APENAS no seu dispositivo atual.

💥 Se você perder o celular ou formatar o computador, perderá TUDO.

🛡️ O backup é SUA única garantia. Faça toda semana!

Como Fazer Backup

  1. 1
    Vá em Configurações
  2. 2
    Clique em "Baixar Backup"

    Isso gerará um arquivo .json seguro com seus dados

  3. 3
    Guarde em local seguro

    Envie para o seu WhatsApp, e-mail ou nuvem pessoal

Restaurar Backup

  1. 1
    Tenha o arquivo .json no aparelho
  2. 2
    Acesse Configurações
  3. 3
    Clique em Restaurar Dados

    Selecione o arquivo baixado e aguarde processar

6

Dúvidas Frequentes (FAQ)

Como faço para trocar de dispositivo?

Para usar em outro aparelho, basta fazer o backup no dispositivo antigo (Configurações > Fazer Backup), enviar o arquivo para o novo e usar a função "Restaurar Dados" nas configurações.

Funciona sem internet?

Sim! O Sistema foi projetado para funcionar offline após instalado ("Instalar App"). Você só precisa de internet para disparar mensagens por WhatsApp para seus clientes.

Como cadastrar novos profissionais/serviços?

No menu principal, acesse a aba "Profissionais" ou "Serviços". Clique no botão com o sinal de mais (+) e preencha as informações necessárias. No caso dos profissionais, defina a % de comissão de cada um para automação.
